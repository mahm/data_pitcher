# DataPitcher

DataPitcher send SQL result data to your google spreadsheets.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'data_pitcher'
```

And then execute:
```bash
$ bundle
```

## Setup

```ruby
bundle exec rails g data_pitcher:install
```
